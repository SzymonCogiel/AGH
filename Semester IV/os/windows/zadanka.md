# POWERSHELL CHEAT SHEET

## [1. Filesystem/PLIKI]()
  
  
 Takie echo/print powershell 
  ```
  Write-Host 'CONTEXT'
  ```
  
  Tworzenie objektu/itemu
  ```
  New-Item -Path "PATH" -Name "NAME" -ItemType "directory/file" 
  ```
 
 Takie cd powershellowe
  ```
  Push-Location $PATH 
  ```
  
  Inne cd powershellowe
  ```
  Set-Location $PATH
  ```
